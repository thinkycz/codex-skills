# Artifact Intake

If the user has not provided the controlling source artifacts yet, ask for the smallest set that would unblock safe planning:

- written client specification path or pasted content
- Figma URL, node link, or screenshots
- Google Stitch project or screen ids
- API contract, Swagger or OpenAPI link, backend docs, or admin requirements

Accept partial inputs and keep moving with what exists. Record missing artifacts explicitly instead of guessing.
