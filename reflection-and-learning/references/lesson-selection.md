# Lesson Selection

Use this reference to decide whether a takeaway is worth preserving.

## Good Lessons

- explain a recurring failure mode
- prevent a class of repeated mistakes
- sharpen verification or review quality
- clarify a repo-specific convention discovered during real work
- improve future routing, decomposition, or closeout decisions

## Weak Lessons

- generic advice the model already knows
- one-off situational trivia
- emotional or motivational summaries with no behavioral effect
- long prose that does not change future decisions

## Rule

Prefer one durable guardrail over five vague observations.
