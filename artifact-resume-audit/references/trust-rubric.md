# Trust Rubric

Use this when judging whether an artifact should drive the restart.

## Trustworthy

- recent enough to reflect the likely current state
- consistent with nearby artifacts
- supported by evidence or explicit status

## Partially Useful

- still informative
- but clearly behind the current state or missing important updates

## Contradictory

- disagrees with other important artifacts in a way that changes the next step

## Obsolete

- no longer useful for restart decisions

## Rule

- Trust the narrowest artifact that is both current enough and evidence-backed.
